# To-Do App (React + Vite)

## Project Overview

This is a simple **To-Do List web application** built using **React** and **Vite**.  
The project demonstrates basic React concepts such as **components, state management (`useState`)**, and **event handling**, along with clean CSS styling.

This repository contains **only the essential source files**.  
The `node_modules` folder is **not included** and must be generated locally.

----------------------------------------------------------------
